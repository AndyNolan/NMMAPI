# Get-NMMAppRoles

## SYNOPSIS
List all App Roles.

## SYNTAX

```
Get-NMMAppRoles [<CommonParameters>]
```

## DESCRIPTION
List all App Roles.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-NMMAppRoles
id                       : {guid}
description              : Full access to all features.
displayName              : Super Admin
level                    : 1
canBeRestrictedByAccount : False
```
